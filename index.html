Step 3

Create Firestore database

Firestore Database → Create database → Start in test mode → Done

Step 4

Get web config keys

Project Settings → Your apps → </> → Register app → copy 6 values

Settings
Configure data sources and your account.

Google Sheet — All Data

Make your sheet public (Share → Anyone with link → Viewer). Create 3 tabs named exactly:

Tab: Terms   → A: Term | B: Definition | C: Remarks
Tab: Laws    → A: Law  | B: Provision | C: Notes
Tab: Questions → A: Category | B: Question | C-F: Choice A-D | G: Answer (A/B/C/D) | H: Remarks
Row 1 in each tab = headers (skipped automatically)

How to get your Sheet ID

1
Open your Google Sheet and look at the URL bar.
2
Copy the long string between /d/ and /edit — that is your Sheet ID.
3
Click Share → Anyone with the link → Viewer → Done.
4
Paste the Sheet ID above → click Save & Fetch All.

Admin Mode

Toggle admin mode to add resource links, manage phases, and add custom lessons. Turn it off when studying so the interface stays clean.

Admin Off
When on: you can add links, phases, and lessons in the Study Tracker.
When off: clean study view — same as what students see.
